# 开发路线图

本文档描述了实时渲染实验室的长期开发计划。

项目目标是通过独立的 Demo 逐步探索现代图形技术，
并基于共享的核心渲染基础设施进行构建。

路线图按阶段划分，各阶段代表渲染技术复杂度的递进。

[English Roadmap](../en/roadmap.md)

---

## 第一阶段 — 框架基础 ✅

目标：建立稳定的图形实验环境。

已完成：

- [x] 应用程序运行时，包含主循环和层栈（Layer Stack）
- [x] 基于 GLFW 的窗口和输入系统（键盘、鼠标、滚轮）
- [x] OpenGL 4.6 渲染循环，采用 DSA（Direct State Access）
- [x] 着色器管理 — 文件加载、多源码、`#type` 预处理、Uniform 缓存
- [x] 基于 STB Image 的纹理加载（R8、RGB8、RGBA8、Depth 格式）
- [x] 基于智能指针的资源管理（`Scope<T>`、`Ref<T>`）
- [x] 帧缓冲抽象 — 多颜色/深度附件、像素回读、尺寸调整
- [x] 透视相机与第一人称调试控制器
- [x] 顶点缓冲/索引缓冲/VAO 抽象及布局描述
- [x] 网格（Mesh）与 MeshFactory（立方体、平面、全屏四边形）
- [x] 材质系统（Shader + 纹理槽绑定）
- [x] ImGui 集成与调试面板（帧率、内存、Demo 选择器）
- [x] Demo 框架 — DemoBase、DemoRegistry、LabLayer，支持热切换
- [x] 日志与文件系统工具
- [x] 单元测试与集成测试基础设施（Google Test）

---

## 第二阶段 — 基础实时渲染

目标：实现基本的渲染技术。

进度：

- [x] 前向渲染 Pass
- [x] 方向光，包含环境光 + 漫反射着色
- [x] 阴影贴图 — 从光源视角的深度 Pass，带偏移修正的阴影比较
- [x] 阴影贴图调试可视化（TexturePreviewPass）
- [x] 多 Pass 场景渲染器（ShadowPass → ForwardPass）
- [x] 3x3 PCF 软阴影
- [x] 阴影 Pass 正面剔除（无需大偏移即可减少 Acne）
- [x] 高光照明（Blinn-Phong）
- [x] CPU 端预计算法线矩阵
- [x] `enum class TextureSlot` 类型安全纹理槽绑定
- [ ] 多光源支持（点光源、聚光灯）
- [ ] 材质参数 UI（逐 Demo 的 ImGui 控件）
- [ ] 天空盒 / 环境贴图渲染
- [ ] 法线贴图
- [ ] 模型加载（OBJ / glTF）

交付物：

- 包含不同光源类型的多个光照 Demo
- 可调参数的基础材质系统
- 效果对比截图

## 第三阶段 — 现代渲染管线

目标：过渡到更先进的渲染管线。

技术方向：

- 延迟渲染与 G-Buffer 架构
- HDR 渲染与色调映射（Reinhard、ACES 等）
- 泛光效果（阈值 + 高斯模糊 + 合成）
- Gamma 校正管线
- 抗锯齿（MSAA、FXAA）

交付物：

- 延迟渲染 Demo
- 前向渲染与延迟渲染的对比
- HDR / 泛光效果 Demo

---

## 第四阶段 — 基于物理的渲染（PBR）

目标：实现基于物理的着色。

技术方向：

- Cook-Torrance BRDF（GGX / Schlick / Smith）
- 金属度 / 粗糙度工作流
- 基于图像的光照（IBL）
- 环境贴图预滤波
- BRDF 积分查找表
- PBR 材质编辑器

交付物：

- PBR 材质球 Demo
- HDR 环境光照

---

## 第五阶段 — 屏幕空间效果

目标：使用屏幕空间技术增强视觉真实感。

技术方向：

- 屏幕空间环境光遮蔽（SSAO）
- 屏幕空间反射（SSR）
- 屏幕空间全局光照（实验性）
- 运动模糊
- 景深

交付物：

- 效果对比 Demo（开/关切换）
- 参数调节工具

---

## 第六阶段 — 程序化几何

目标：探索算法驱动的内容生成。

技术方向：

- 基于噪声的地形生成（Perlin、Simplex）
- 带 LOD 的程序化地表
- 体素地形实验
- 基于区块（Chunk）的世界表示
- Marching Cubes / Dual Contouring

交付物：

- 程序化地形 Demo
- 支持实时参数编辑的交互式地形可视化

---

## 第七阶段 — GPU 技术

目标：利用现代 GPU 编程技术。

技术方向：

- 计算着色器（Compute Shader）
- GPU 粒子系统
- GPU 视锥 / 遮挡剔除
- 间接渲染（MultiDrawIndirect）
- GPU 驱动管线

交付物：

- GPU 粒子模拟 Demo
- 性能对比实验

---

## 第八阶段 — 光线追踪实验

目标：探索基于光线的渲染技术。

技术方向：

- CPU 光线追踪参考渲染器
- BVH 加速结构
- 路径追踪与重要性采样
- 光栅化 + 光线追踪混合渲染

交付物：

- 基础路径追踪器
- 光线追踪可视化工具

---

## 第九阶段 — 前沿研究课题

未来可能探索的方向：

- 体积渲染（雾、云、上帝光线）
- 全局光照方法（RSM、LPV、体素锥追踪）
- 神经渲染实验
- 实时 / 离线混合技术
- 虚拟阴影贴图
- 网格着色器（Mesh Shader）

这些领域属于探索性质，可能随时间演变调整。
